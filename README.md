## ✨ What is this?

This repository powers a **personal portfolio experience** — a digital space where design meets engineering.
It’s lightweight, expressive, and intentionally crafted to feel less like a static page and more like a story.

> Think: **developer portfolio** × **interactive canvas** × **personal brand laboratory**.

---

## 🧠 Philosophy

I build with three principles:

- **Clarity over clutter** → every section should have a purpose.
- **Motion with meaning** → animations should guide attention, not distract.
- **Personality in every pixel** → code should feel human.

## 🚀 Run Locally

```bash
# 1) Clone
git clone https://github.com/<your-username>/Geetansh-Jangid.git

# 2) Enter the folder
cd Geetansh-Jangid

# 3) Open in browser
# (double click index.html)
```

Or with a local server:

```bash
npx serve .
```
